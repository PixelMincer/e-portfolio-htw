My Capabilities

If I had to define my capabilities here's what I would come up with. During my apprenticeship in Computer Science I had the possibility to learn various different programming languages. Some of them I learned during my free time and some during work. Currently I can program in the following languages:

  • C / C++
  • C# / Java
  • Html / CSS
  • JavaScript / TypeScript

I soon became interested in developing video games. It always made me wonder how to implement certain things such as gravity, water reflection etc. in games. Turns out today it is not that hard to implement these kind of things with the help of Game Engines.

As I am currently working in IT which is an ever-evolving field you always have room for improvement. And that's what I find important. If I would know everything I would be pretty bored. For me learning and experimenting with new technology is a great joy which I would not trade for anything in the world.
